dentro del archivo se adjuntan los archivos de android studio, la api y la base de datos.

1. el archivo se android se ejecuta con normalidad en el IDE

2. para hacer el uso de la api se debe de agregar la carpeta prueba a Xampp en la carpeta de htdocs, la ruta debe de quedar algo similar a esto:
C:\xampp\htdocs\prueba\APIs%Android
con el fin de que android la pueda consumir sin problema alguno.

3. levantar el servicio de mysql ya que se usa una base de datos web por la cantidad de datos que almacena la empresa.
